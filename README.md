# ElectoGuide 🗳️

**ElectoGuide** is a premium, AI-powered interactive platform designed to empower citizens with comprehensive knowledge about the election process. Built with a focus on modern design aesthetics and user-centric functionality, it serves as a one-stop assistant for navigating democracy.

![ElectoGuide Hero](hero-banner.png)

## 🌟 Key Features

- **Gemini AI Assistant**: An intelligent chatbot powered by Google Gemini to answer complex questions about voting and electoral laws.
- **Candidate Arena**: Side-by-side comparison tool to evaluate candidates' visions, roles, and key policies.
- **Voter's Pulse**: Real-time mock poll simulation with live sentiment analysis and trends.
- **Find My Polling Station**: A search tool to locate assigned voting booths with mock map integration.
- **Interactive Timeline**: A phase-by-phase journey of the election process from announcement to results.
- **Eligibility Checker**: A quick tool to assess voter eligibility and required documentation.
- **Multilingual Support**: Available in English, Español, and हिन्दी.
- **Voter Readiness Checklist**: A persisted checklist to ensure you're fully prepared for election day.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- A [Gemini API Key](https://aistudio.google.com/app/apikey) (required for the AI Assistant)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/UDaygupta12512/Election-PromptWars.git
   cd Election-PromptWars
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env` file in the root directory and add your Gemini API Key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the application:**
   ```bash
   npm start
   ```
   Open your browser and navigate to `http://localhost:3000`.

## 🎨 Design Philosophy

ElectoGuide utilizes a **premium glassmorphism** aesthetic with:
- Dynamic animated particle backgrounds.
- Harmony-curated color palettes (Deep Purples, Cyans, and Teals).
- Smooth CSS transitions and micro-animations.
- Fully responsive layout for all device types.

- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Backend**: Node.js, Express.js
- **AI**: Google Gemini 1.5 Flash API
- **Icons/Graphics**: Lucide-inspired SVG icons & Custom Assets
